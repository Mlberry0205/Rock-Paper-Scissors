# Rock-Paper-Scissors

A classic two-person game against the computer! Choose “rock, paper, scissors, then shoot!” Rock crushes scissors, scissors cut paper, and paper covers rock. Want more of a challenge?  Add in an alien and lizard!


## Technologies Used

- JavaScript
- CSS
- HTML
